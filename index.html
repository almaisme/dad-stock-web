爸專用看盤台
輸入代號 → 一鍵開資料來源 → 照流程看盤
例：2330、0050、日勝生
小提醒
先用「代號」最準。若輸入名稱，會改用搜尋頁。
固定看盤順序（爸版）
  1. 看大盤方向：今天偏多 / 偏空 / 盤整？（先不要逆勢）
  2. 看量：成交量有沒有放大？是不是假突破？
  3. 看均線：5/10/20 日線有沒有糾結 → 翻多排列？
  4. 看關鍵價位：前高、缺口、支撐壓力、整數關卡
  5. 設定停損停利：先想好再進（不要進了才想）
  6. 分批：不追高、不梭哈、照紀律做
你的紀律（快速版）
  • 單筆最大虧損 ≤ 總資金 5%
  • 進場前設定停利/停損，成交後不亂改
  • 不追高、不報復交易、不因連賺提高風險
之後要加強的功能（你說一聲就做）
  • 接 API：輸入代號 → 回傳「三線合一」判斷與風控提醒
  • 我的自訂清單：爸爸常看的股票一鍵點
  • 一鍵產生「今天操作計畫」：進場/停損/分批停利
